## ℹ️ About the Website
**MIRA.CO** represents a modern, highly interactive e-commerce landing page themed around an Autumn/Winter knitwear collection in London. It is built as a responsive, single-page application using HTML5, Vanilla CSS, and modern JavaScript, styled with clean layouts and Bootstrap 5 utilities.

The design features a dark, luxurious color palette (warm tones, pale browns, and dark greys), custom typography (Playfair Display and DM Sans), and polished micro-interactions aimed at delivering an immersive user experience.

---

## ⚙️ What the Website Does
This website integrates several interactive developer features and shopping modules:
1. **Interactive Cursor Tracking**: Features a custom cursor point and ring follower that dynamically resizes and glows when hovering over interactive elements (links, buttons, and product cards).
2. **Shopping Cart & Wishlist System**: A client-side state manager handles adding products to the shopping bag, updating badges, recalculating totals in a sliding overlay drawer, and toggling product wishlist status.
3. **Full-Stack API Simulator**: Mock-fetches data from a simulated MongoDB database and dynamically creates product cards while outputting timing metrics (e.g., `200 OK — Server responded in 124ms`).
4. **JS Playground Console**: Evaluates raw JavaScript inputs directly in the browser and displays execution states or catch block error messages.
5. **Dynamic Theme Switcher**: Allows quick switching of UI color combinations dynamically.
6. **Scroll-Driven Animations**: Leverages the `IntersectionObserver` API to trigger smooth fade-in animations as elements enter the viewport.

## 🛠️ What Can Be Done (User Activities)
Visitors on the site can interact with the following components:
* **Browse the Autumn/Winter Edit**: Scroll through clean grid layouts featuring seasonal knitwear and basics.
* **Add Items to Cart**: Click product options to slide open the cart drawer, see order itemized summaries, and remove items dynamically.
* **Toggle Wishlists**: Save items for later by toggling heart icons, which triggers real-time visual alerts.
* **Simulate Backend Operations**: Run the API Simulation in the dev console card to visualize how a real product fetch works with server logs.
* **Run Custom JavaScript**: Input any valid JS code inside the interactive playground textarea and click "Run Code" to view immediate outputs or troubleshoot errors in real time.
* **Test Mobile Responsiveness**: Collapse the browser to check out the animated hamburger navigation drawer and responsive layout adjustments.
